# Work Timer
작업 집중을 위한 개인용 웹 타이머입니다.
간단한 타이머 기능과 함께 메모, 시간 기록, 이미지/문장 커스터마이징 기능을 제공합니다.

## 주요 기능
### 타이머 & 스톱워치
- 작업 시간을 측정합니다. 측정한 시간을 기록하여 제목을 붙이고 관리할 수 있습니다.

### 커스터마이징
- 타이머 제목, 이미지 등을 커스텀할 수 있습니다.
- 이미지는 jpg, png, gif까지 지원하며, 여러 장 등록하면 자동으로 슬라이드되어 움짤이 됩니다.
- 이미지를 클릭하면 비율이 전환됩니다. (잘림/전체보기)
- 랜덤 텍스트 출력 기능을 지원합니다. 원하는 문장들을 등록하면, 등록된 문장 중 하나가 2분에 한 번씩 무작위로 출력됩니다.

### 메모
- 간단한 메모 기능입니다. 메모 카드를 클릭하면 색상이 변합니다. 세 가지 색상을 지원합니다.

### 캡처
- 기록/메모 영역을 캡처할 수 있습니다. 캡처 시 전체 기록/메모 영역이 이미지로 저장됩니다.

## 데이터 저장
- 모든 데이터는 브라우저의 **Local Storage**에 저장됩니다.
- 새로고침 / 창 종료 후에도 데이터가 유지됩니다.
- 단, 다른 브라우저나 기기에서는 공유되지 않습니다.

## 사용 방법
1. 페이지 접속
2. 타이머 또는 스톱워치 사용
3. 메모 및 기록 작성
4. 설정 아이콘을 통해 이미지/텍스트 커스터마이징 가능

## 주의사항
- 시크릿 모드에서는 데이터가 저장되지 않습니다.
- 브라우저 캐시를 삭제하면 데이터가 사라질 수 있습니다.
